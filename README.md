# SocketProgrammingPractice
計算機網路 socket Programming 作業

## 內容
基於TCP的基礎聊天室

## code 大綱
server.py: Server端負責處理&接收所有client端的連線與訊息  
client.py: terminal 上執行client端  
clientGUI.py: 一個簡單的視窗app 來執行client端的畫面
